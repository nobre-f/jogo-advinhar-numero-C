Um jogo simples feito em linguagem C onde o jogador precisa adivinhar um número aleatório gerado pelo computador.

📌 Sobre o projeto

Este projeto foi desenvolvido para praticar conceitos básicos da linguagem C, como:

Entrada e saída de dados (scanf e printf)
Estruturas de repetição (while)
Condicionais (if/else)
Geração de números aleatórios (rand e srand)
🎮 Como funciona
O programa sorteia um número de 1 a 20
O jogador tenta adivinhar o número
O sistema dá dicas:
🔼 "Muito alto"
🔽 "Muito baixo"
O jogo continua até o jogador acertar
No final, mostra o número de tentativas
